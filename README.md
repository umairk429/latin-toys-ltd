# LATIN-TOYS BI Dashboard

This Power BI dashboard was created as part of a Junior BI Analyst Developer task.

# Objectives
The report answers key business questions:

- Trend in website sessions and order volume
- Session-to-order conversion rate and its trend
- Most successful marketing campaigns
- Revenue per order and revenue per session trends
- Highest and lowest selling products

# Dashboard Structure

# Page 1 — Executive Overview
Shows:
- KPI summary (Revenue, Sessions, Orders, Conversion Rate)
- Traffic and order trends
- Conversion rate trend
- Revenue efficiency trends

# Page 2 — Marketing & Product Performance
Shows:
- Campaign performance (traffic vs efficiency)
- Campaign performance table
- Top and lowest selling products

# Tools & Techniques
- Power BI
- DAX measures
- Data modeling & relationships
- Time-based analysis

# Files Included
- Power BI report (.pbix)
- SQL queries for analysis
